# AI Emergency Surgery Command Center

## Overview
AI Emergency Surgery Command Center is a futuristic single-page full-stack web application that simulates a hospital surgery monitoring dashboard. It allows doctors to monitor critical patient vitals, surgery status, and AI-generated alerts in real time through an interactive dashboard interface.

The project demonstrates modern web development workflow including frontend development, backend integration, database design, version control, CI/CD pipeline, and deployment.

---

## Features

- Emergency surgery monitoring dashboard
- AI diagnosis simulation
- Surgery countdown timer
- ECG heartbeat animation
- ICU patient vitals graph
- Critical alerts monitoring
- Real-time dashboard analytics

---

## Technology Stack

### Frontend
- HTML

### Backend
- Node.js

- MySQL

## CI/CD Pipeline

Continuous Integration and Continuous Deployment are implemented using **GitHub Actions**.

The pipeline automatically:

- checks code when changes are pushed
- installs dependencies
- runs testing and linting
- prepares the project for deployment

This ensures reliable and automated software delivery.

---

## Deployment

The frontend dashboard is deployed using **Netlify**, which provides free hosting for static web applications.

Live Website:
https://projectbyhemananda.netlify.app

Whenever code is pushed to GitHub, Netlify automatically redeploys the website.

---

## How the System Works

1. The user accesses the dashboard using a web browser.
2. The frontend displays patient monitoring panels and AI alerts.
3. JavaScript simulates real-time patient vitals and surgery updates.
4. Backend services handle patient data and API communication.
5. MySQL database stores patient and surgery information.
6. GitHub manages the source code.
7. CI/CD pipelines ensure automated testing and deployment.
8. Netlify hosts the live application.

This workflow demonstrates a modern full-stack web development pipeline.
